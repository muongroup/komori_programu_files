# シミュレーションデータの解析等で使用するプログラムのまとめ

## 各プログラム名と使用用途の説明


### cope_dump_binalcustom_t.cpp
- シミュレーションのダンプデータを解析し各ベクトルプロットごとのカウント数を導出するプログラム

### cope_dump_binalcustom45do.cpp
- シミュレーションのダンプデータを解析し各ベクトルプロットごとのカウント数を導出するプログラム (検出面を45°回転させている)

### cope_dump_binalcustom45do_forloop.cpp
- シミュレーションのダンプデータを解析し各ベクトルプロットごとのカウント数を導出するプログラム (検出面を45°回転させている)
- 解析するエネルギー帯を連続的に変えながら解析したいときに使用する


### cope_dump_binalcustom45do_xhalf.cpp
- シミュレーションのダンプデータを解析し各ベクトルプロットごとのカウント数を導出するプログラム (検出面を45°回転させている)
- 検出面の解像度をx方向で半分にしている

### cope_dump_binalcustom45do_yhalf.cpp
- シミュレーションのダンプデータを解析し各ベクトルプロットごとのカウント数を導出するプログラム (検出面を45°回転させている)
- 検出面の解像度をy方向で半分にしている

### cope_dump_binalcustom45do_xyhalf.cpp
- シミュレーションのダンプデータを解析し各ベクトルプロットごとのカウント数を導出するプログラム (検出面を45°回転させている)
- 検出面の解像度をxy方向で半分にしている

### atten_hist31.cpp
- 減衰率分布を作成するプログラム

### atten_hist31_xhalf.cpp
- 減衰率分布を作成するプログラム
- x方向の解像度半分

### atten_hist31_yhalf.cpp
- 減衰率分布を作成するプログラム
- y方向の解像度半分

### atten_hist31_xyhalf.cpp
- 減衰率分布を作成するプログラム
- xy方向の解像度半分

### comparehist.cpp
- 計測データの比を取るプログラム


### comparehist_xhalf.cpp
- 計測データの比を取るプログラム
- x方向の解像度半分

### comparehist_yhalf.cpp
- 計測データの比を取るプログラム
- y方向の解像度半分

### comparehist_xyhalf.cpp
- 計測データの比を取るプログラム
- xy方向の解像度半分
